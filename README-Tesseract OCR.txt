Hướng dẫn cài đặt Tesseract trên windows:
- Download bộ cài tại đây: https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-setup-3.05.02-20180621.exe
- Sau khi cài đặt, cho đường dẫn tới  folder Tesseract-OCR vào biến môi trường Path
- Copy file vie.traineddata vào folder Tesseract-OCR/tessdata
- Cuối cùng, cài đặt package pytesseract cho python
